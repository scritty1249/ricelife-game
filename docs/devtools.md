# Development tools
Tools can be downloaded locally, or visiting from Github Pages as a static browser page from the `/tools` directory.

Terrian creator tool, for drawing new maps, can be found at [`/tools/terrain-creator`](https://scritty1249.github.io/ricelife-game/ricelife-game/tools/terrain-creator).

Terrain renderer tool, for generating map thumbnails and previews, can be found at [`/tools/terrain-renderer`](https://scritty1249.github.io/ricelife-game/ricelife-game/tools/terrain-renderer).
